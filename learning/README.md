# DermaAI Learning Roadmap

Welcome to your personalized learning journey! This roadmap is designed to take you from a beginner to an intermediate full-stack developer using your own project, **DermaAI**, as the classroom.

### Phase 1: Foundation (Weeks 1-2)
*   **Module 1: The Big Picture** - [module_01_foundation.md](./module_01_foundation.md)
    *   Concepts: HTTP, Client-Server, Docker, Network Topology.
    *   Goal: Understand how the pieces fit together.

### Phase 2: Backend Fundamentals (Weeks 3-5)
*   **Module 2: FastAPI & Requests**
    *   Concepts: Routing, Pydantic Validation, Dependency Injection.
    *   Goal: Build robust API endpoints.
*   **Module 3: Data Modeling**
    *   Concepts: SQLAlchemy, Relationships, Migrations (Alembic).
    *   Goal: Design and manage your database schema.

### Phase 3: Authentication & Security (Week 6)
*   **Module 4: Securing the App**
    *   Concepts: JWT, Hashing, Middleware, CORS.
    *   Goal: Protect your users and data.

### Phase 4: Frontend Fundamentals (Weeks 7-9)
*   **Module 5: React & Next.js Core**
    *   Concepts: Components, Props, Hooks, SSR vs CSR.
    *   Goal: Build reusable UI components.
*   **Module 6: State & Data Fetching**
    *   Concepts: Zustand, React Query, Forms.
    *   Goal: Manage application state effectively.

### Phase 5: Advanced Patterns (Weeks 10-12)
*   **Module 7: Async & Background Tasks**
    *   Concepts: Celery, Redis, Event Loops.
    *   Goal: Handle long-running processes (like AI analysis).
*   **Module 8: Testing & Quality**
    *   Concepts: Pytest, Vitest, E2E.
    *   Goal: Write code with confidence.

### Phase 6: Integration & DevOps (Weeks 13-14)
*   **Module 9: Production Readiness**
    *   Concepts: CI/CD, Monitoring, Optimization.
    *   Goal: Prepare for deployment.
